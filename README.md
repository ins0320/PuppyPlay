# 🐶 PuppyPlay 🙋

### 📢 프로젝트 설명
* 기존 애견유치원 사이트에서 제공되지 않는, 서비스에 아쉬움을 느껴 개발
* 애견유치원 이용 고객을 대상으로 제공되는 서비스
* 월별 일정 공지, 하루 일과 안내, 공지사항, Q&A 기능 제공
